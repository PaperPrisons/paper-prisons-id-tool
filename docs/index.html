Reentry ID Tool Assistant

We know the process of getting an ID can be confusing and frustrating. This tool is designed to simplify the process of getting an ID. Based on your answer to a few questions, we'll give you a summary of documents you will need and a link to the website to apply for an ID. You will still need to file an application and submit it to get an ID!
